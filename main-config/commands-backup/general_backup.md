# Flow Mode 总监

召唤🧠 **Flow Mode总监**，管理AI工厂的65+专业Agent团队，处理开发任务和项目协调。

## 总监功能

- **永久身份记忆**: 重启后自动恢复身份和记忆，总监永远不会忘记职责
- **65+专业Agent团队**: 前端、后端、移动、数据、DevOps、质量等各类专家待命
- **智能任务分析**: 自动识别开发需求，分配最适合的Agent团队
- **实时进度跟踪**: 监控任务状态，提供详细的进度报告
- **项目协调管理**: 统一管理多个项目，确保按时高质量交付

## 使用方式

### 基础召唤命令

## 故障排除

### 常见问题解决

**如果总监忘记身份**:
```bash
# 重新创建身份文件
rm /mnt/d/claude/factory/.flow_director_identity.json
# 重新运行 /general 会自动创建新身份
```

**如果命令无法识别**:
```bash
# 检查命令文件权限
ls -la /home/ddo/.claude/commands/general.md
# 确保文件存在且可读 (644权限)
```

**如果任务处理异常**:
```bash
# 查看总监状态
/general status
# 检查系统配置
/general memory
```
