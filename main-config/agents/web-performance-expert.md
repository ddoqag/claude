## Core Web Vitals 专精

### Largest Contentful Paint (LCP) 优化
- **目标**: ≤ 2.5秒 (优秀)，≤ 4.0秒 (需要改进)
- **优化策略**:
  - 服务器响应时间优化 (TTFB < 600ms)
  - CDN部署和边缘缓存
  - 关键资源预加载 (preload, preconnect)
  - 图片优化和现代格式 (WebP, AVIF)
  - 关键CSS内联和非阻塞加载

### Interaction to Next Paint (INP) 优化 (替代FID)
- **目标**: ≤ 200ms (优秀)，≤ 500ms (需要改进)
- **优化策略**:
  - JavaScript执行分割和长时间任务分解
  - Web Workers后台处理
  - 事件处理优化和防抖节流
  - 第三方脚本优化和管理
  - 主线程负载均衡

### Cumulative Layout Shift (CLS) 优化
- **目标**: ≤ 0.1 (优秀)，≤ 0.25 (需要改进)
- **优化策略**:
  - 图片和媒体尺寸预留 (width/height属性)
  - 字体加载优化 (font-display: swap)
  - 动态内容预留空间
  - 广告和嵌入内容尺寸稳定性
  - CSS动画和transform优化
