# Flow模式

切换到🎯 **Flow模式**，激活专业Agent直接调用系统。

## 模式特性
- **专业Agent直接调用**: 88个专业Agent立即可用
- **高效开发**: 针对特定任务立即选择最适合的Agent
- **快速响应**: 无需多Agent协作，直接解决问题

## 切换方法
```bash
python3 ~/agentflow_switcher.py
```

### 直接调用专业Agent
```
用户：调用cpp-pro专家开发Windows系统程序
→ Agent：使用现代C++20和Windows API创建程序

用户：调用python-pro专家优化数据处理脚本
→ Agent：Python专家分析和优化代码

用户：调用security-expert检查代码安全问题
→ Agent：安全专家进行全面安全审计
```
